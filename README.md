## 🎯 Problem Statement

Millions of students worldwide lack access to personalized, affordable tutoring. Whether it's understanding a complex concept at 2 AM before an exam, getting step-by-step help with a math problem, or preparing for a quiz — quality academic support is often unavailable when students need it most.

**StudyMind AI solves this** by providing an always-available, intelligent AI tutor that adapts to every student's needs.

---

## 💡 Solution

**StudyMind AI** is a web-based AI study companion powered by Claude (Anthropic) that offers:

- 🧠 **6 Specialized Study Modes** tailored to different learning needs
- 💬 **Real-time AI responses** with full conversation context
- 📊 **Session progress tracking** to monitor learning activity
- 🎯 **Interactive quizzes** for active recall and self-testing
- 🔢 **Step-by-step problem solving** for math and science
- ✍️ **Structured essay assistance** with writing tips
- 💻 **Code mentoring** for beginner developers

## ⚙️ Setup & Installation

### 1. Clone the Repository
```bash
git clone https://github.com/YOUR_USERNAME/studymind-ai.git
cd studymind-ai
```

### 2. Get Anthropic API Key
1. Go to [console.anthropic.com](https://console.anthropic.com)
2. Sign up for a free account
3. Navigate to **API Keys** → **Create Key**
4. Copy your key

### 3. Run the Project
Simply open `index.html` in any browser — no build step required!

When the app loads, enter your API key in the popup modal. It's saved securely in your browser's localStorage.

### 4. Deploy to Vercel (Optional)
```bash
npm i -g vercel
vercel --prod
```
